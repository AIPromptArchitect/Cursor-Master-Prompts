# 🎯 Cursor Master Prompts

> **Open-source `.cursorrules` templates for AI-powered development.**

Welcome to the AI Prompt Architect open-source templates. These are static `.cursorrules` examples designed to give you a head start when working with AI-powered coding assistants like Cursor, Claude, and Copilot.

## 📁 Templates

| Template | Use Case |
|----------|----------|
| `react-nextjs.cursorrules` | React + Next.js full-stack apps |
| `python-fastapi.cursorrules` | Python backends with FastAPI |
| `rust-axum.cursorrules` | Rust web services with Axum |
| `typescript-node.cursorrules` | TypeScript + Node.js APIs |

> **Coming soon:** More framework-specific templates for Vue, Svelte, Go, and more.

## 🚀 Usage

1. Copy the relevant `.cursorrules` file into your project root
2. Open your project in Cursor
3. The AI assistant will automatically follow the rules defined in the file
